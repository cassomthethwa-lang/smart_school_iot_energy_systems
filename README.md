# Smart School IoT & Energy Monitoring System

## Overview
This project focuses on designing a Smart School system that uses IoT devices, smart switches, and solar integration to monitor and control electrical usage across the school.

My role in this project is centered on Data Science — collecting, analysing, and interpreting electricity usage data to improve energy efficiency and enable intelligent control of devices.

## Objectives
- Monitor electricity usage of devices and classrooms
- Identify patterns of energy waste using data analysis
- Enable data-informed remote control of electrical devices
- Compare solar energy input with electricity consumption
- Improve overall energy efficiency of the school

## My Data Science Role
- Converting electrical activity into datasets
- Analysing usage patterns using Python
- Designing logic for smart control based on data trends
- Supporting decision-making using collected data

- Data Analysis
- IoT Concepts
- Energy Monitoring
- Problem Solving
